What?

Gems/Sass — Gems Over Sass — or GOS is a simple Micro-Framework that helps you when you start coding up your Web Designs; using a set of collected and implemented Haml , Sass , and Javascript open-source files and libraries to get the job done faster and easier.

It’s based on HTML 5 Boilerplate , and powered by Sass. Plus tons of other features that will help you develop impressive Web Applications.

Download v1.x.x Documentation

Why ?

Strengthened by Bourbon

The beautiful, simple and lightweight mixin library for Sass.

HTML 5 Boilerplate

The base styles of the Framework is based on HTML5 Boilerplate; the most popular Frontend Template on the Web.

Powered by Sass

It has been established that SASS is the most mature and powerful CSS extension to date.

1 Site in 1 File

Build all of your final CSS into 1 file. For even faster performance.

Multi-Colored Templates

Just define your class names and the color HEX values and you’re good to go.

RTL Support

Implement your RTL styles effortlessly. Aside from the bundled RTL grid system.

Web Fonts Made Easier

Easilty embed your fonts and use a shorthand for each font which will save you lots of work hours.
